Roadmap

PoC 進度

Phase 0環境與基線

GPU、NATS、schema、語言護欄完成。

Phase 1STT → Bus

串流 STT、partial/final、replay 測試完成。

Phase 2MT 平面

vLLM 翻譯、防抖、單語失敗隔離完成。

Phase 3TTS 平面

CosyVoice2 串流合成與 route_by 完成。

Phase 4LiveKit 整合

已完成 TTS 發軌；下一步是 mic ingress、viewer 與 floor control。

Phase 5高扇出與插拔

gateway、惰性 TTS、插件插入與壓測。
