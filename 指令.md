# 指令.md — 操作 runbook

> 開什麼、怎麼開、怎麼測都在這裡；開發規則與坑在《規則.md》，任務在《代辦.md》。
> 所有指令都在 repo 根目錄（/nas-data/allen-speakin）下執行。

## 完整 demo 啟動順序（兩台裝置對講）

```bash
# 1. 基礎服務（通常已常駐，先檢查）
docker start speakin-nats              # NATS（JetStream;監控埠 :8222）
bash deploy/dev/livekit.sh             # LiveKit SFU :7880（已在跑會自動略過/報已存在）
bash deploy/dev/vllm_mt.sh             # vLLM 翻譯 :8001（載入需 ~1-2 分鐘）
curl -s localhost:8001/v1/models       # 出現 breeze-7b-mt 才算就緒

# 2. 管線（STT ingress + MT + TTS 發軌;三個「就緒」都印出才算起完）
.venv/bin/python -m services.runner --pipeline pipelines/default.yaml --session ses_dev

# 3. gateway（token + 字幕事件 WS 橋）
bash deploy/dev/gateway.sh             # :8800
curl -s localhost:8800/api/health      # {"ok":true}

# 4. viewer 前端
cd frontend && npm run dev             # :5173（--host，LAN 可連）
curl -s localhost:5173/api/health      # 確認 vite 代理 → gateway 通

# 5. 對外 https 通道（手機/外部裝置要開麥克風必須走 https）
cloudflared tunnel --url http://localhost:5173
# 輸出裡的 https://xxx.trycloudflare.com 就是測試網址（每次重開會換）
```

裝置端：開 tunnel 網址 → 取不同名字、選不同語言 → 加入 → 開麥克風。
注意：tunnel 只載頁面/字幕/信令；**聲音（WebRTC/UDP）同網段直連才通**，
真外網裝置需 TURN（LiveKit Cloud / 自架，見《代辦.md》）。

## 各服務單獨操作

```bash
# NATS
docker start speakin-nats / docker stop speakin-nats
curl -s localhost:8222/varz | head    # 監控

# LiveKit
bash deploy/dev/livekit.sh            # 起
docker stop speakin-livekit           # 停

# vLLM（不要手起，固化的環境解法都在腳本裡）
bash deploy/dev/vllm_mt.sh

# runner（管線;--nodes 只起部分節點、--set 覆寫任意 config）
.venv/bin/python -m services.runner --pipeline pipelines/default.yaml --session ses_dev
.venv/bin/python -m services.runner --pipeline pipelines/default.yaml --session dev \
  --nodes stt,mt --set stt.source=tests/replay/golden/zh_demo.wav --set stt.language=zh
# ↑ --set stt.source=<wav> 即檔案回放模式（不走 LiveKit ingress）

# gateway
bash deploy/dev/gateway.sh            # :8800

# 終端字幕（除錯用訂閱端）
.venv/bin/python -m services.viewer --session ses_dev

# cloudflared 通道
pgrep -af cloudflared                 # 看狀態
pkill cloudflared                     # 關
cloudflared tunnel --url http://localhost:5173   # 開（網址重開會換）
```

## 測試

```bash
# 無外部依賴（CI 同款）
.venv/bin/ruff check .
.venv/bin/python -m pytest tests/unit -q

# 重播/整合測試（標注所需服務;CI 不跑）
.venv/bin/python -m pytest tests/replay                    # 全部（需 GPU+NATS+vLLM+LiveKit）
.venv/bin/python -m tests.replay.test_stt_replay           # GPU+NATS：STT 檔案回放
.venv/bin/python -m tests.replay.test_livekit_stt          # GPU+NATS+LiveKit：mic ingress
.venv/bin/python -m tests.replay.test_livekit_tts          # GPU+NATS+LiveKit：TTS 發軌
.venv/bin/python -m pytest tests/replay/test_e2e_stt_mt.py # GPU+NATS+vLLM：stt→mt e2e
```

## 前端

```bash
cd frontend
npm install                            # 首次 / lockfile 變更後
npm run dev                            # viewer 開發伺服器 :5173
npm run typecheck && npm run build     # CI 同款檢查
npm run gen:types                      # schema 變更後重生 packages/sdk/src/events.gen.ts
```
